# AI Content Generator

## Project Overview
The AI Content Generator is a web-based application designed to create AI-generated content such as text, captions, blog ideas, product descriptions, and marketing content. The project demonstrates the use of Artificial Intelligence and Prompt Engineering to improve productivity and automate content creation tasks.

This project was developed as part of the Week 2 Individual Project in the AI Bootcamp 2026 program.

---

## Objectives
- Build a functional AI-powered content generation tool
- Apply prompt engineering techniques
- Demonstrate AI-assisted productivity solutions
- Create a portfolio-ready AI application

---

## Features
- Generate AI-written content instantly
- User-friendly interface
- Multiple content generation options
- Custom prompts support
- Fast and efficient content creation
